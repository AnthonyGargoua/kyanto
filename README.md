## 📖 Présentation du Projet

**Kyanto** est un projet de design interactif explorant les concepts de minimalisme et de navigation immersive. Ce site vitrine met l'accent sur l'harmonie entre le contenu textuel et le vide visuel, offrant une expérience utilisateur (UX) apaisante et haut de gamme.

📍 **Lien du projet :** [https://kyanto.vercel.app/](https://kyanto.vercel.app/)

### 🎯 Objectifs Créatifs
- **Épure Visuelle** : Utilisation d'une palette de couleurs restreinte pour focaliser l'attention sur les éléments clés.
- **Fluidité Interactive** : Mise en œuvre de micro-interactions pour enrichir le parcours utilisateur sans le surcharger.
- **Modernité Mobile** : Une approche "Fluid Design" où les éléments s'adaptent organiquement à chaque format d'écran[cite: 1].

---

## 🛠 Stack Technique

- **Frontend Core** : HTML5 sémantique et CSS3 avancé (Grid Layout & Flexbox).
- **Interactivité** : JavaScript ES6 pour la gestion des événements et des états de navigation[cite: 1].
- **Animations** : Transitions CSS personnalisées et GSAP pour le séquençage fluide des apparitions[cite: 1].
- **Performance** : Optimisation des assets pour un score de chargement ultra-rapide sur Vercel[cite: 1].

## 🎨 Philosophie de Design

Kyanto repose sur le concept du "Less is More". Chaque marge, chaque police et chaque espace blanc a été pensé pour créer une structure aérée :
- **Variables CSS** : Centralisation des couleurs et espacements pour une cohérence parfaite sur tout le site[cite: 1].
- **Mobile-First** : Priorisation de l'ergonomie tactile avec des zones de clic larges et intuitives[cite: 1].
